Repositorio de git

git status: me proporciona informacion sobre el estado del repositorio. Rama actual, listado de commits, archivos de estado 'Untracked', archivos en Staging area git add -A : me permite enviar todos los archivos en estado 'untracked' hacia el staging area git add 'nombrearchivo': envia solamente el archivo indicado al staging area git commit -m "Mensaje del commit" git push --force origin main
